SQLite 缓存层
NetCDF 4D → 按 time_step 切片 → zlib 压缩 → SQLite BLOB；惰性解压单时间步；源文件签名校验自动重建；WAL 模式 + 线程锁防并发冲突。
PRAGMA journal_mode=WAL; -- 并发读取优化
