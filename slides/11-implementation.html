模块化架构与 Cesium 解耦

module.js 注册即拓展
在配置文件中声明模块信息，启动器自动生成入口按钮，新增功能模块只需添加一行配置，即可注册到系统中。
{ id: 'iso', title: 'IsoSurface', component: IsoSurfaceModule, policy: {x:0, y:52} }
CesiumAPIProvider 安全解耦
将 Viewer 实例封装为 5 个独立子 API（相机/瓦片/点击/标注/GeoJSON），模块只能通过接口操作地图，杜绝全局污染与误操作。
createCesiumApi({viewer, rendererRef, entitiesRef, handlerRef, ...})
useWindowManager 浮动窗口
统一处理窗口打开、层叠、拖拽、吸附、最大化与最小化；关闭逻辑由模块通过 cleanup 自定义，资源问题会被限定在对应模块内并显式暴露。
registerCleanup(module.id, () => module.releaseResources())
