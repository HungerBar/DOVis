# Poetry GIS：诗词时空 GIS 知识平台

## 项目概述

Poetry GIS 是一个融合 WebGIS、自然语言处理（NLP）与大语言模型（LLM）的诗词时空分析平台。

项目以中国古典诗词为核心对象，通过对诗词文本进行空间语义解析，将传统文学内容转化为可进行 GIS 分析的时空数据，并基于 WebGIS 实现文化空间可视化、时空统计分析与 AI 地图叙事生成。

系统核心目标包括：

* 诗词文本的空间化表达
* 文化语义的 GIS 组织与分析
* 诗人行为与文化传播的时空研究
* 基于自然语言的专题地图自动生成
* AI 驱动的沉浸式地图漫游体验

项目整体定位为：面向数字人文的诗词时空 GIS 分析平台。

# 一、诗词时空语义数据库

系统通过 NLP 与 LLM 技术，从古诗词文本中自动提取：

* 地点
* 时间
* 诗人
* 朝代
* 情感
* 意象
* 空间场景
* 地理路线

等信息，并完成：

* 地理编码（Geocoding）
* 时空结构化
* 语义标签化
* GIS 空间索引构建

最终形成支持 GIS 空间分析与 AI 检索的诗词时空数据库。


## 1.1 空间语义解析

系统对诗词文本进行语义结构化处理。

### 地名解析

自动识别诗词中的空间实体，例如：

* 长安
* 姑苏
* 黄鹤楼
* 洞庭湖
* 玉门关

并完成：

* 古今地名映射
* 坐标定位
* GIS 空间入库

用于后续空间分析与地图表达。


### 时间语义解析

提取：

* 朝代
* 创作时期
* 季节
* 时辰

例如：

* 春
* 秋
* 夜
* 暮春
* 中秋

并构建诗词时间索引。


### 情感与意象识别

系统对诗词中的文化语义进行分类与标签化。

例如：

| 类型 | 示例          |
| -- | ----------- |
| 情感 | 思乡、离别、豪放、忧国 |
| 意象 | 月、酒、孤舟、长江   |
| 场景 | 边塞、水乡、夜泊、登楼 |

形成支持空间统计分析的文化语义标签体系。


# 二、GIS 时空分析

系统基于 ArcGIS / GeoScene 的空间分析能力，对诗词数据进行时空可视化与空间统计分析。


## 2.1 诗人迁徙轨迹分析

根据：

* 诗词创作地点
* 创作时间
* 诗人生平信息

重建诗人的历史活动轨迹。

例如：

苏轼：
眉山 → 汴京 → 黄州 → 惠州 → 儋州

系统支持：

* 时间轴动态播放
* 路径动画
* 历史底图切换
* 不同时期轨迹对比


## 2.2 文化意象空间分布分析

系统支持对特定文化意象进行空间统计与专题制图。

例如：

“月”意象在唐诗中的空间分布。

系统可生成：

* 热力图（Heat Map）
* 核密度图（Kernel Density）
* 聚类分析结果
* 分朝代专题图

用于分析：

* 不同地域的文化偏好
* 意象空间聚集现象
* 文学文化中心分布


## 2.3 诗词空间密度分析

基于诗词空间点数据，系统生成：

* 点密度图
* Kernel Density
* 文化热点地图

用于分析：

* 江南诗词高密度区域
* 唐宋文化中心迁移
* 文学空间分布差异


# 三、AI 地图漫游生成

AI 地图漫游生成是系统的核心功能之一。

该模块尝试实现：从自然语言描述自动生成诗词专题地图。

## 3.1 自然语言输入

用户可输入：

* “想看关于江南秋夜离别的诗词”
* “李白关于月亮与长江的作品”
* “唐代边塞诗路线”
* “适合雨夜阅读的江南诗词”

系统自动解析用户意图。


## 3.2 LLM 语义解析

大语言模型提取用户输入中的语义条件。

例如：

输入：

> “想看关于江南秋夜离别的诗词”

系统解析：

| 类型 | 内容   |
| -- | ---- |
| 地域 | 江南   |
| 季节 | 秋    |
| 情感 | 离别   |
| 场景 | 夜    |
| 意象 | 月、孤舟 |

并生成结构化 GIS 查询条件。


## 3.3 时空数据库检索

系统基于诗词时空语义数据库进行检索，筛选符合条件的：

* 地点
* 诗词
* 诗人
* 文化场景

例如：

* 苏州
* 杭州
* 秦淮河
* 扬州

并构建候选空间节点集合。


## 3.4 GIS 路线生成与地图组织

系统结合：

* 空间邻近关系
* 历史交通关系
* 文化关联度
* 时间顺序

通过 GIS 排序与路径组织算法，自动生成：

> “诗词文化路线”。

同时自动完成：

* 底图选择
* 地图缩放层级调整
* 主题图层配置
* 空间要素组织

形成完整的专题地图场景。


## 3.5 AI 地图叙事生成

系统最终生成：

* 地图飞行动画
* 时间轴
* 诗句弹窗
* 空间路线漫游
* AI 文化解说文本
